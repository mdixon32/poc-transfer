# BloodBikesReactApp

A minimal Expo-based React Native app with a Home screen and bottom tab bar.

## Requirements
- Node.js (16+ recommended)
- npm (or yarn)
- Expo CLI (optional)

## Install

From project root:

```bash
npm install
# or
# yarn
```

Then install native dependencies for navigation (recommended):

```bash
npx expo install react-native-gesture-handler react-native-screens react-native-safe-area-context
npm install @react-navigation/native @react-navigation/bottom-tabs
```

## Run

Start the dev server:

```bash
npm start
# or
# npx expo start
```

Open on iOS Simulator / Android Emulator from the Expo dev tools, or run:

```bash
npm run ios
npm run android
```

Notes:
- This project uses Expo for simplicity. If you prefer a bare React Native project, let me know and I can scaffold that instead.
